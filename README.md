# point-system-demo
포인트 시스템 데모

## 기술 스택

| 기술              | 버전  | 설명              |
|-----------------|-----|-----------------|
| Java            | 21  | 프로그래밍 언어        |
| Spring Boot     | 3.x | 웹 애플리케이션 프레임워크  |
| Spring Data JPA | -   | ORM 및 데이터 접근 계층 |
| H2 Database     | -   | 인메모리 데이터베이스     |
| Redis           | -   | 분산락 (동시성 제어)    |
| Redisson        | -   | Redis 분산락 클라이언트 |
| JUnit 5         | -   | 테스트 프레임워크       |
| Gradle          | -   | 빌드 도구           |
